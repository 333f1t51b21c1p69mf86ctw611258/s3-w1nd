Overview of Modeling File Editor:

 

                       

                       

Modeling File Editor helps users create a new or modify an existing modeling file with ease. Modeling File Editor has a rich set of TCL command templates which users can use as a starting point to create or modifying a modeling file.

Modeling File Editor provides two ways to create or modify a modeling file:

·        Auto script

·        Manual script

On Auto script tab, users can utilize a set of predefined actions and TCL command templates to create a modeling file.

On Manual script tab, users will manually create or modify the modeling file as we do with notepad editor.
