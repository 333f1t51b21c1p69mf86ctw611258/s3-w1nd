How to create a new Modeling File

 

Steps to create a new modeling file:

 

1.      Click on Action button on Auto tab to open a predefined modeling file action dialog.

2.      Select init_action as the first action for the modeling file. ( for a modeling file to work properly, the first action must always be init_action) Close the dialog box.

Or

      Select New from the menu bar

               

 

3.      Click on Add button to open a predefined TCL command template dialog.

 

         

4.      Select any appropriate TCL command for the modeling file.

5.      Fill in inputs for a command if necessary.

6.      Click on Apply to append the selection to the Editor.

      

7.      Select other commands if necessary.

8.      Click OK to close the dialog box.

9.      Repeat from step 1 for other actions and commands.

10.  After creating process completed, select Save from the Menu bar to save the modeling file.

 

Any changes made in a modeling file on Auto Script will be reflected to Manual Script immediately and vice versa.

 

 How to modify a Modeling File

 

Steps to modify a modeling file:

 

  1. Open a modeling file from the menu bar
  2. Click on Modify or Delete button to edit a modeling file action. Once an action deleted, so are its TCL commands.
  3. Click on Add button to open a predefined TCL command template dialog.
  4. Select any appropriate TCL command for the modeling file.
  5. Fill in inputs for a command if necessary.
  6. Repeat step 2 if necessary.
  7. After modifying process completed, select Save from the Menu to save the file.
