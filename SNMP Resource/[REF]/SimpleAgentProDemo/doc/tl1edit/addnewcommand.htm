To Add New Command:

 

            This dialog is provided to add new command in the .tl1 file. The user can press ‘Ok’ button when the entries from the dialog is filled properly or ‘Cancel’ to cancel the action.

 

           

 

 

            Once the command is added the user can modify or delete the command.

The user can add Acknowledgement, Response or Autonomous message in the command by selecting from the edit  menu, tool bar or from the popup menu.
