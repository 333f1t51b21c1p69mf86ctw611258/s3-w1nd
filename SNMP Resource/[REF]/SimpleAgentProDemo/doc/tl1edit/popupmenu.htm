Node Popup Menu:

 

The node popup menu will consist of the following items.  The functionality of each of these items is exactly the same as the corresponding menu option or tool bar items.  The node popup menu simply provide a quicker way for the user to invoke that function.  These items will be disabled / enabled in the same manner as the menu option or the tool bar items. 

           

Following are the popup menu items:

 

  • Cut: To cut the selection from 
  • Copy: To copy the selection.
  • Paste: To paste the text from the clipboard.
  • Add Acknowledgement: To add the acknowledgement message in the command.
  • Add Response: To add the response message in the command.
  • Add Autonomous Message: To add the autonomous message in the command.
