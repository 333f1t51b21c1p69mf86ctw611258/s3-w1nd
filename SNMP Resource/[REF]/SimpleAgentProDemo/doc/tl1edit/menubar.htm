Menu Bar:

 

Following are the main menu items available for the user.

           

File Menu:     

           

The File Menu Contains Following items.

New: The user can choose New or Ctrl+N to create new TL1 file.

Open: Open menu item or Ctrl+O can be used to open an existing file.

Save: Save or Ctrl+S can be used to save the file which is currently opened.

Save As : This menu item can be selected to save the opened file with some other name.

Exit: Exit or Ctrl+Q can be used to quit from the editor.

Edit Menu:    

                   

The Edit Menu Contains Following items.

Cut: To cut the selection from the editor to the clipboard, Ctrl+X can be used.

Copy: To copy the selection from the editor to the clipboard, Ctrl+C can be used.

Paste: To paste in the editor from clipboard , Ctrl+V can be used.

Find: To find the command. Ctrl+F can be used instead.

Help Menu:   

                                  

The Help Menu Contains Following items.

Index: Help index, F1 can be used to see the index.

About Tl1 Editor: About dialog of Tl1 Editor.
