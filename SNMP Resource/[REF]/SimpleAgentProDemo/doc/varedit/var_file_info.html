Variable file tree contains following types of nodes :

Address Node

These are the non-leaf-nodes excluding the Table Nodes.

Denoted as   on the node.

Available Operations: Create Instance

Whenever the right button is clicked on a node, a context sensitive Popup Menu pops up.        

Create Instance: Whenever new instance is to be created this option is used .

 Newly created instance will have the OID as a name , and will be added as a scalar node at appropriate position in the tree.

If only  node is selected, then Create Instance menu item is enabled.

If multiple nodes are selected then all menu items on Popup Menu and EditMenu  are disabled.

Scalar Node

Following table shows Scalar Nodes in the variable tree.

    Notation   

Scalar Node Type

       

Exists in .var and .cmf file with read-only access.

       

Exists in .var and .cmf file with read-write access.

       

Exists in only in .cmf file with read-only access.

       

Exists in only .cmf file with read-write access.

Available Operations : Create Modify Instance , Cut ,Copy ,Paste

Functionality: Whenever the right button is clicked on a node, a context sensitive PopupMenu pops up.

If only one scalar node is selected, then Create Modify Instance, Cut ,Copy ,Paste menu items are enabled.

If multiple nodes are selected then  if  they are of same type then only Cut ,Copy ,Paste on menu items PopupMenu and EditMenu  are enable . Otherwise all are disabled.

Table Node

Denoted as  on the node.

Available  Operations   : Dynamic Row Template, Create Modify Instance , Cut ,Copy and Paste.

Whenever the right button is clicked on a node, a context sensitive PopupMenu pops up.

If only one table node is selected, then Create Modify Instance, Cut ,Copy ,Paste and Dynamic Row Template menu items are enabled.

For table node, Create Modify Instance invokes Table View Dialog.

If multiple table nodes are selected then only Cut ,Copy ,Paste on menu items PopupMenu and Edit Menu  are enable . Otherwise all are disabled.

Entry Node

These nodes are the immediate childs of the Table nodes.

Denoted as  on the node below Table Node.

Available Operations :  Create Modify Instance, Cut ,Copy ,Paste and Dynamic Row Template

Whenever the right button is clicked on a node, a context sensitive PopupMenu pops up.

If only one Entry node  is selected, or a single Entry node as well as it’s parent table Node are selected  then Create Modify Instance,  Cut ,Copy ,Paste and Dynamic Row Template menu items are enabled.

For Entry node, Create Modify Instance  invokes Table View Dialog.

Note:  If multiple entry nodes or entry node along with table node belonging to other entry node are selected then, only Cut ,Copy ,Paste on menu items PopupMenu and EditMenu  are enable.Otherwise all are disabled.

Columnar Node

These are the children of Entry Nodes indirectly children of table nodes.

A Column node shows how many rows are present for that column. This would be the same as the columns in the table to which this column belongs.

Following table shows Columnar Nodes in the variable tree.

    Notation   

Columnar Node Type

       

Exists in .var and .cmf file with read-only access.

        

Exists in .var and .cmf file with read-write access.

         

Exists in only in .cmf file with read-only access.

        

Exists in only .cmf file with read-write access.

Available Operations : Create Modify Instance , Cut ,Copy ,Paste

Whenever the right button is clicked on a node, a context sensitive PopupMenu pops up.

If only one column node is selected, then Create Modify Instance, Cut ,Copy ,Paste menu items are enabled.

 Create Modify Instance  for column node invokes Column View Dialog.

It allows to modify column node values.

If multiple column nodes are selected then  if  they are of same type then only Cut ,Copy ,Paste on menu items PopupMenu and EditMenu  are enable . Otherwise all are disabled.

Instance Node

These nodes are leaf nodes for variables defined in the .var file but do not exist in the MIB file.

Denoted as  on the node.

The behavior is exactly like the scalar nodes.

Available  Operations :

Whenever the right button is clicked on a node, a context sensitive PopupMenu pops up.

If only one instance node is selected, then Create Modify Instance , Cut ,Copy , Paste menu items are enabled.

If multiple nodes are selected then  if  they are of same type then only Cut ,Copy ,Paste on menu items PopupMenu and EditMenu  are enable . Otherwise all are disabled.
